# PRT(PeerReviewTemplate)

각 항목을 스스로 확인하고 체크하고 확인하여 작성한 코드에 적용하세요.

- [ ] 코드가 정상적으로 동작하고 주어진 문제를 해결했나요?
- [ ] 주석을 보고 작성자의 코드가 이해되었나요?
- [v] 코드가 에러를 유발할 가능성이 있나요?
- [v] 코드 작성자가 코드를 제대로 이해하고 작성했나요? (직접 인터뷰해보기)
- [v] 코드가 간결한가요?
